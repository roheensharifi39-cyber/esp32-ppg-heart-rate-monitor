# ESP32 Pulse Monitoring and Signal Analysis

Built a real-time heart rate monitoring system using an ESP32 and MAX30102 photoplethysmography sensor. Implemented beat detection and BPM calculation on-device, displayed results via an OLED interface, and streamed physiological data to Python for visualization, CSV data collection, statistical analysis, and machine learning–based detection of anomalous pulse patterns.

## Features
- Real-time BPM calculation on ESP32
- OLED heart-rate display
- Python live plotting
- CSV data logging
- BPM trend analysis
- ML-based pulse anomaly detection

## Tech Stack
- ESP32
- MAX30102
- OLED display
- Arduino / PlatformIO
- Python
- Pandas
- Matplotlib
- Scikit-learn
